# Selectionsort Algorithm

The Selectionsort algorithm is an easy but instable sort algorithm.

## Complexity

- Time Complexity (Best) = O(n²)
- Time Complexity (Average) = O(n²)
- Time Complexity (Worst) = O(n²)

Space Complexity: O(1)
